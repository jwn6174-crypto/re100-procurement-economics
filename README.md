# RE100 이행수단 유형별 경제성 분석

기업 유형별로 RE100 이행수단의 경제성 순위를 매기고, **단가 기준 순위**와 **위험·적격성을 반영한 종합순위**가 어떻게 달라지는지(역전, flip) 분석하는 연구용 코드입니다.

> 개인연구 (진행 중)

## 연구 질문

기업의 전력소비·물리적 자산(자가발전 여력)·제도적 조건(ETS 편입)이 다를 때, RE100 이행수단(녹색프리미엄·REC·제3자 PPA·직접 PPA·자가발전)의 경제성 순위는 유형별로 어떻게 달라지며, 단가 기준 순위와 위험·적격성 보정 순위는 어떤 유형에서 뒤집히는가?

## 핵심 발견

가장 싼 수단이 최선은 아니다. 단가로는 하위권인 직접 PPA가, ETS 편입 기업이 글로벌 공급망 압박을 받을 때 최상위권으로 역전된다. 그리고 이 역전은 입력값을 흔들어도(몬테카를로 5000회 기준 ETS 기업 직접 PPA의 1~2위 유지 확률 약 97~98%) 무너지지 않는다.

## 기업 유형 (업종 + 규모)

3축(소비규모 · ETS 편입 · 자가발전 커버리지)으로 7개 대표 유형을 정의한다. 자가발전은 "가능/불가"가 아니라 **부지면적과 수요로 산출한 커버리지 비율(0~1)**로 모델링한다.

| 유형 | 연소비(GWh) | ETS | 부지면적(㎡) | 자가발전 커버리지 |
|------|-----------|-----|------------|----------------|
| 반도체 대기업 | 5,000 | ○ | 1,920,000 | 1.5% |
| 철강 대기업 | 4,000 | ○ | 30,850,000 | 29.2% |
| 석유화학 대기업 | 3,500 | ○ | 4,587,561 | 5.0% |
| 자동차 완성차 대기업 | 2,000 | ○ | 7,327,348 | 13.9% |
| 데이터센터 | 1,500 | ✗ | 100,000 | 0.3% |
| 중소 제조(부품) | 5 | ✗ | 30,000 | 22.7% |
| 영세 서비스·소매 | 3 | ✗ | 0 (임차) | 0.0% |

부지면적은 대표 기업 공개 자료 기반(포스코·현대차·LG화학·SK하이닉스). 커버리지는 가용부지 비율 0.24(한국에너지공단 산업단지 태양광 잠재량: 지붕 47.5% × 설치가능 50%)·이용률 15%·설치계수 0.12kW/㎡로 산출. 반도체와 철강은 둘 다 대기업·부지 보유이나 수요 대비 부지 비율 차이로 커버리지가 정반대(1.5% vs 29.2%)인 점이 이 모델의 핵심이다.

## 분석 단계

| 단계 | 모듈 | 내용 |
|------|------|------|
| Phase 1 | `cost_engine.py` | 5개 수단을 동일 기준(원/kWh)으로 환산하는 비용엔진. 총비용/증분비용 두 모드. 자가발전은 커버리지 비율 × 자가단가 + (1−비율) × 보완수단으로 혼합. |
| Phase 2 | `archetypes.py` | 3축(소비규모·ETS·자가발전 커버리지)으로 7개 기업 유형 정의, 특성을 엔진에 연결. |
| Phase 3 | `scoring.py` | 비용 외 4기준(가격안정성·계약유연성·ETS적격성·RE품질) 점수화 + AHP 가중치 + 종합순위. |
| Phase 4 | `robustness.py`, `breakeven.py` | 몬테카를로·토네이도·임계점으로 역전의 강건성 검증. |

## 실행

```bash
python -m venv .venv
.venv\Scripts\activate        # Windows
pip install numpy pandas matplotlib

python src/archetypes.py      # 유형별 순위 매트릭스
python src/scoring.py         # 위험·적격성 보정 + 역전 분석
python src/robustness.py      # 강건성 검증
python src/breakeven.py       # 임계점 분석
```

## 방법론 핵심 결정

- **비용엔진(LCOE)**: 균등화발전비용 방식(IRENA·IEA·NREL 표준), 국내 적용 신경철 외(2024).
- **PPA 단가**: 비공개 협상가이므로 발전사업자 기회비용(SMP+REC, 바닥)~산업용 전력량요금(천장) 구간으로 처리.
- **두 비용 모드**: 총비용(조달 총액)과 증분비용(일반요금 대비 추가분)을 구분.
- **자가발전 커버리지**: 부지면적·수요로 산출. "부지 있다/없다"의 흑백이 아니라 수요 대비 발전 가능량 비율로 처리.
- **ETS적격성**: 제도가 인정/불인정 이진이므로 0/1로 처리.
- **다기준 종합**: AHP(Saaty 1980)로 가중치 산출(일관성비율 CR<0.1 검증), 시나리오(비용중시/안정성중시/공급망대응)별로 분기.
- **강건성**: 몬테카를로·토네이도·임계점 분석(기술경제성 평가 표준 기법).

## ETS 편입 기준

배출권거래법 제8조 — 최근 3년 연평균 업체 125,000 tCO₂ 이상 또는 사업장 25,000 tCO₂ 이상이면 할당대상. 직접배출 기준이므로 반도체·철강·석유화학·완성차(현대차=대상, 감사보고서 확인)는 편입, 전력은 많이 쓰나 직접배출이 작은 데이터센터·소규모 기업은 비편입.

## 한계 (값 교체·문헌·보완 예정)

분석 **골격**은 완성되었고 자가발전 커버리지 구조 보완도 반영했다. 아래는 근거 있는 값으로 교체·정밀화할 항목이다.

**값 교체·문헌**
- 소비규모(절대 GWh): 순위는 한전 업종별 전력사용량 근거이나 절대값은 대표 대기업 스케일의 추정. 기업별 실측·공시로 교체.
- 중소 제조·데이터센터 부지·소비: 공개 자료 부재로 추정값.
- RE품질 점수: 판단값 → GHG Protocol Scope 2 8개 품질기준 체크리스트 충족 개수로 재산정.
- 가격안정성·계약유연성 점수: 문헌 근거 보강.
- AHP 가중치: 잠정값 → RE100 실무자·연구자 쌍대비교 설문으로 산출.
- 망 이용료·태양광 CAPEX: 한전 요금표·사업자 견적·KEEI LCOE로 교체.

**구조 정밀화**
- 자가발전 RE품질·안정성 점수가 현재 "100% 자가발전" 가정. 실제 커버리지 비율을 반영하도록 정밀화 예정.
- PPA 협상력 채널: 현재 고REC 역전으로 잠복.

**가정·범위 (명시)**
- **영세 서비스·소매의 직접 PPA '불가' 처리**: 제도상 금지가 아니라 현실적 단순화다. 직접 PPA 발전설비 1MW 초과 요건은 2025.7 On-Site의 경우 폐지됐고, 사용자 요건(제3자 PPA 기준 300kW ≈ 연 1GWh 내외)은 소형 기업도 충족 가능하다. 그럼에도 '불가'로 둔 것은 임차 점포 기반 영세 사업자의 부지 부재·협상·계약 부담을 반영한 가정이다.
- **기업 특성–관점(가중치) 연결**: 어떤 기업이 어떤 관점(비용중시/안정성중시/공급망대응)을 갖는지는 본래 기업 대상 설문으로 실증해야 한다. 그러나 본 연구는 3주 일정의 팀 프로젝트 제약상 이를 수행할 수 없어, 기업 특성(ETS 편입·수출 비중·전력비 민감도)과 관점의 연결을 합리적 추론에 근거해 제시한다. 실증적 검증은 후속 과제로 남긴다.
