# RE100 이행수단 유형별 경제성 분석

기업 유형별로 RE100 이행수단의 경제성 순위를 매기고, **단가 기준 순위**와 **위험·적격성을 반영한 종합순위**가 어떻게 달라지는지(역전, flip) 분석하는 연구용 코드입니다.

> 국민대학교 CLIM Lab · 개인연구 (진행 중)

## 연구 질문

기업의 전력소비·자본·물리적 자산·제도적 조건이 다를 때, RE100 이행수단(녹색프리미엄·REC·제3자 PPA·직접 PPA·자가발전)의 경제성 순위는 유형별로 어떻게 달라지며, 단가 기준 순위와 위험·적격성 보정 순위는 어떤 유형에서 뒤집히는가?

## 핵심 발견

가장 싼 수단이 최선은 아니다. 단가로는 하위권인 직접 PPA가, ETS 편입 기업이 글로벌 공급망 압박을 받을 때 최상위로 역전된다. 그리고 이 역전은 입력값을 흔들어도(몬테카를로 5000회 기준 92.5~100%) 무너지지 않는다.

이는 국내 RE100의 실제 사용 통계(녹색프리미엄 56.9%·PPA 4% 미만)와 선호 통계(직접 PPA 27.4%로 선호 1위)의 괴리를, 단가(사용)와 위험·적격성(선호)의 분리로 설명한다.

## 분석 단계

| 단계 | 모듈 | 내용 |
|------|------|------|
| Phase 1 | `cost_engine.py` | 5개 수단을 동일 기준(원/kWh)으로 환산하는 비용엔진. 총비용/증분비용 두 모드. |
| Phase 2 | `archetypes.py` | 3축(소비규모·부지·ETS)으로 5개 기업 아키타입 정의, 특성을 엔진에 연결. |
| Phase 3 | `scoring.py` | 비용 외 4기준(가격안정성·계약유연성·ETS적격성·RE품질) 점수화 + AHP 가중치 + 종합순위. |
| Phase 4 | `robustness.py`, `breakeven.py` | 몬테카를로·토네이도·임계점으로 역전의 강건성 검증. |

## 실행

```bash
python -m venv .venv
.venv\Scripts\activate        # Windows
pip install numpy pandas matplotlib

python src/archetypes.py      # 유형별 순위 매트릭스
python src/scoring.py         # 위험·적격성 보정 + 역전 분석
python src/robustness.py      # 강건성 검증
python src/breakeven.py       # 임계점 분석
```

## 방법론 핵심 결정

- **PPA 단가**: 비공개 협상가이므로, 발전사업자 기회비용(SMP+REC, 바닥)~산업용 전력량요금(천장) 구간으로 처리.
- **두 비용 모드**: 총비용(조달 총액)과 증분비용(일반요금 대비 추가분)을 구분.
- **ETS적격성**: 제도가 인정/불인정 이진이므로 0/1로 처리.
- **다기준 종합**: AHP로 가중치 산출(일관성비율 CR<0.1 검증), 시나리오(비용중시/안정성중시/공급망대응)별로 분기.

## 한계 (값 교체·구조 보완 예정)

현재 분석 **골격**은 완성되었으나, 아래는 근거 있는 값으로 교체하거나 구조를 보완해야 한다.

**값 교체**
- 소비규모: 임시값 → 한전 업종별 전력사용량으로 실측 대표값 도출
- RE품질 점수: 판단값 → GHG Protocol Scope 2 8개 품질기준 체크리스트 충족 개수로 재산정
- 가격안정성·계약유연성 점수: 문헌 근거 보강 필요
- AHP 가중치: 잠정값 → RE100 실무자·연구자 쌍대비교 설문으로 산출
- 망 이용료·태양광 CAPEX: 한전 요금표·사업자 견적·KEEI LCOE로 교체

**구조 보완 (코드 수정)**
- 자가발전을 on/off가 아닌 **수요 커버리지 비율**로 모델링 (반도체처럼 수요가 거대한 경우 대응)
- PPA 협상력 채널 활성화 (현재 고REC 역전으로 잠복)
